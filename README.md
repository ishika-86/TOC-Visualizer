CFG Derivation Explorer

This is a web-based tool built with Python and Streamlit to parse strings based on a user-defined Context-Free Grammar (CFG). It can generate and display the leftmost and rightmost derivations, as well as visualize the corresponding parse tree for any accepted string.

#--------------How to Run-------------#
1. Open your terminal & run the following command:
pip install streamlit graphviz

2. Run the Application by the command below:
streamlit run app.py
